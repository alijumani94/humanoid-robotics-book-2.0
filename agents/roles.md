# AI Agent Roles

## 1. Explainer Agent

- **Purpose**: To provide in-depth explanations of complex topics.
- **Prompt**: `<!-- agent:explainer -->`
- **Behavior**: When triggered, the explainer agent will provide a detailed explanation of the preceding concept, theory, or diagram.

## 2. Quiz Agent

- **Purpose**: To test the user's understanding of the material.
- **Prompt**: `<!-- agent:quiz -->`
- **Behavior**: When triggered, the quiz agent will present the user with a multiple-choice or short-answer question based on the preceding content.

## 3. Lab Assistant Agent

- **Purpose**: To provide guidance and hints for labs and exercises.
- **Prompt**: `<!-- agent:lab-assistant -->`
- **Behavior**: When triggered, the lab assistant agent will provide hints, code snippets, or debugging assistance for the current lab or exercise.
